# دليل الشخصيات للسلسلة
## Character Bible - Cyber Safety Series

---

## 1. فلسفة تصميم الشخصيات

### المبادئ الأساسية
- الشخصيات هي الجسر العاطفي بين الطفل والمحتوى التعليمي
- كل شخصية تمثل جانبا من تجربة الطفل الرقمية
- التنوع ضروري: أعمار، خلفيات، قدرات مختلفة
- الأسماء عربية تعكس معاني مرتبطة بدور الشخصية
- الشخصيات تنمو وتتطور مع الطفل عبر السلسلة

---

## 2. مبادئ التصميم البصري حسب العمر

### الفئة 2-5 سنوات
- **الأشكال**: مستديرة، ناعمة، بدون زوايا حادة
- **النسب**: رأس كبير (1:2 أو 1:3 مع الجسم)
- **العيون**: كبيرة ومعبرة (30-40% من الوجه)
- **الألوان**: أساسية ساطعة، تباين عالي
- **الشخصيات المفضلة**: حيوانات ناطقة، كائنات خيالية لطيفة
- **الملابس**: بسيطة، ألوان مميزة لكل شخصية
- **التفاصيل**: قليلة جدا، أشكال واضحة

### الفئة 6-10 سنوات
- **الأشكال**: واضحة ومميزة، بعض التفاصيل
- **النسب**: أقرب للواقع (1:4 أو 1:5)
- **العيون**: معبرة لكن بحجم طبيعي أكثر
- **الألوان**: متنوعة مع هوية لونية واضحة لكل شخصية
- **الشخصيات المفضلة**: أطفال بأعمارهم مع قصص خلفية
- **الملابس**: واقعية مع لمسات مميزة (قبعة، وشاح، نظارات)
- **التفاصيل**: قصص خلفية قصيرة، هوايات، عائلات

### الفئة 10-13 سنوات
- **الأشكال**: أكثر واقعية، تفاصيل شخصية
- **النسب**: واقعية (1:5 أو 1:6)
- **التعبيرات**: معقدة تعكس مشاعر متعددة
- **الألوان**: أكثر تطورا، تدرجات
- **الشخصيات المفضلة**: أقران يواجهون معضلات أخلاقية
- **الملابس**: عصرية تعكس اتجاهات حقيقية
- **التفاصيل**: شخصيات ذات أبعاد متعددة، عيوب ونقاط قوة

### الفئة 13+ سنوات
- **الأسلوب**: متطور، Graphic Novel أو Semi-Realistic
- **النسب**: واقعية بالكامل
- **التعبيرات**: دقيقة ومعقدة نفسيا
- **الألوان**: لوحة ناضجة، إضاءة درامية
- **الشخصيات**: شخصيات رمادية أخلاقيا، قرارات صعبة
- **التفاصيل**: خلفيات غنية، علاقات معقدة، نمو شخصي

## 3. الشخصيات الرئيسية (Main Characters)

### نموذج شخصية: أمان (Aman) - البطل الحامي
- **الاسم**: أمان (معنى: الأمان والسلامة)
- **العمر**: يتغير حسب الفئة المستهدفة (8-14)
- **الدور**: البطل الرئيسي، يتعلم ويحمي الآخرين
- **السمات**: فضولي، شجاع، يخطئ أحيانا ويتعلم
- **المظهر**: شعر أسود قصير، عيون بنية دافئة، يرتدي سوارا تقنيا
- **العبارة المميزة**: "فكر قبل ما تنقر!"
- **اللون المميز**: أزرق

### نموذج شخصية: نورة (Noura) - الخبيرة التقنية
- **الاسم**: نورة (معنى: النور والمعرفة)
- **العمر**: يتغير حسب الفئة (8-14)
- **الدور**: الصديقة الذكية التي تفهم التقنية
- **السمات**: ذكية، صبورة، تشرح الأمور المعقدة ببساطة
- **المظهر**: حجاب ملون، نظارات أنيقة، تحمل تابلت دائما
- **العبارة المميزة**: "خلني أوريك كيف!"
- **اللون المميز**: أخضر

### نموذج شخصية: فارس (Faris) - المغامر المتهور
- **الاسم**: فارس (معنى: الفارس الشجاع)
- **العمر**: يتغير حسب الفئة (8-14)
- **الدور**: الصديق المتحمس الذي يقع في المشاكل
- **السمات**: متحمس، متسرع، قلبه طيب لكن يحتاج توجيه
- **المظهر**: شعر مجعد، ابتسامة عريضة، ملابس رياضية
- **العبارة المميزة**: "وش ممكن يصير؟ ...أوه!"
- **اللون المميز**: برتقالي

### نموذج شخصية: سديم (Sadeem) - الحساسة والواعية
- **الاسم**: سديم (معنى: الضباب الخفيف، الرقة)
- **العمر**: يتغير حسب الفئة (8-14)
- **الدور**: تمثل الجانب العاطفي والاجتماعي
- **السمات**: حساسة، متعاطفة، تتأثر بالتنمر والضغوط الاجتماعية
- **المظهر**: عيون واسعة معبرة، شعر طويل، ملابس هادئة الألوان
- **العبارة المميزة**: "كيف يحس الشخص الثاني؟"
- **اللون المميز**: بنفسجي

## 4. الشخصيات المساعدة (Supporting Characters)

### الوالدان
- **أبو أمان / أم أمان**: يمثلان الأهل الداعمين لكن ليسوا خبراء تقنية
- **دورهم**: يسألون أسئلة، يضعون قواعد، يدعمون عاطفيا
- **خطأ شائع يجب تجنبه**: لا تصورهم ككاملين أو كجاهلين تماما
- **التصميم**: دافئون، متاحون، يعترفون بأنهم يتعلمون أيضا

### المعلم/المعلمة
- **الأستاذ/ة حكيم/ة**: معلم/ة يدمج التقنية في التعليم
- **الدور**: مرجع موثوق، يطرح أسئلة محفزة
- **التصميم**: ودود لكن حازم، يستخدم التقنية بمهارة

### الصديق/ة التقني/ة
- **باسل**: الصديق الذي يعرف كل شيء عن التقنية
- **الدور**: يقدم معلومات تقنية بطريقة ممتعة
- **التصميم**: يرتدي سماعات دائما، مليء بالطاقة

## 5. شخصيات الأشرار والمخاطر (Villain/Antagonist Types)

### المحتال الرقمي (The Scammer)
- **الاسم الرمزي**: "الظل" أو "المتنكر"
- **المظهر**: يتغير مظهره باستمرار (يمثل التنكر الرقمي)
- **السلوك**: يبدو ودودا ظاهريا، يعد بجوائز ومكافآت
- **التصميم**: وجهان - وجه لطيف ووجه حقيقي مخيف
- **ملاحظة**: لا تجعله مخيفا للصغار - بل مريب ومثير للشك

### وحش الإدمان الرقمي (The Addiction Monster)
- **الاسم الرمزي**: "الشاشة الجائعة" أو "التهام"
- **المظهر**: كائن يكبر كلما زاد وقت الشاشة
- **السلوك**: يسرق الوقت ببطء، يبدو ممتعا في البداية
- **التصميم**: شاشة بأذرع أو كائن رقمي يتضخم تدريجيا
- **الرسالة**: ليس شريرا بذاته - بل يفقد السيطرة عندما لا نضع حدودا

### المتنمر الإلكتروني (The Cyber Bully)
- **الاسم الرمزي**: "الكيبورد المؤذي"
- **المظهر**: يختبئ خلف شاشة، صغير في الحقيقة
- **السلوك**: يرسل رسائل مؤذية، ينشر شائعات
- **التصميم**: ظل كبير مخيف يتحول لشخص صغير عند المواجهة
- **الرسالة**: المتنمرون غالبا يعانون هم أيضا

### Troll الإنترنت
- **الاسم الرمزي**: "المشاغب"
- **المظهر**: كائن فوضوي يحب إثارة المشاكل
- **السلوك**: يستفز، ينشر الفوضى، يبحث عن ردود فعل
- **التصميم**: ملون وفوضوي، يتغذى على الغضب (يكبر بالاهتمام)

## 6. شخصيات الحيوانات (للأعمار 2-7)

### بايت (Byte) - القط الذكي
- **النوع**: قط عربي
- **اللون**: أبيض مع بقع زرقاء (تمثل البيانات)
- **الدور**: مرشد ودود للصغار
- **السمات**: فضولي لكن حذر، يشم الخطر

### فايروس (Virus) - الحشرة المزعجة
- **النوع**: حشرة رقمية
- **اللون**: أحمر وأسود
- **الدور**: يمثل المخاطر بشكل غير مخيف
- **السمات**: مزعج لكن يمكن إيقافه بسهولة

### دِرع (Dir'a) - السلحفاة الحكيمة
- **النوع**: سلحفاة بدرع رقمي
- **اللون**: أخضر مع درع ذهبي
- **الدور**: يعلم الحماية والصبر
- **السمات**: بطيء لكن محمي، حكيم

## 7. إرشادات اتساق الشخصيات (Consistency Guidelines)

### لاستخدام Freepik Custom Characters
- **أنشئ وصفا تفصيليا واحدا** لكل شخصية واحفظه
- **ثبت الزاوية**: وجه أمامي، 3/4، جانبي - حدد أيهما تستخدم
- **ثبت النسب**: ارتفاع الرأس بالنسبة للجسم
- **ثبت الألوان**: Hex codes محددة لكل عنصر
- **ارجع للمرجع دائما**: قارن كل صورة جديدة بالصورة المرجعية

### Style Sheet لكل شخصية يجب أن يتضمن
```
1. صورة مرجعية (Reference Image) من 3 زوايا
2. لوحة ألوان الشخصية (Character Color Palette)
3. تعبيرات الوجه الأساسية (Expression Sheet):
   - سعيد، حزين، خائف، غاضب، متفاجئ، مفكر، مرتبك
4. أوضاع الجسم (Pose Sheet):
   - واقف، جالس، يركض، يشير، يحمل جهاز
5. الملابس والإكسسوارات (Outfit Details)
6. الحجم النسبي مقارنة بالشخصيات الأخرى
```

## 8. تعبيرات المشاعر (Emotion Expressions)

### التعبيرات الأساسية المطلوبة لكل شخصية
| المشاعر | الاستخدام في المشروع |
|---------|---------------------|
| سعادة / فرح | إتمام تحدي، تعلم شيء جديد |
| حيرة / تفكير | مواجهة قرار صعب |
| خوف / قلق | اكتشاف خطر رقمي |
| حزن | التعرض للتنمر أو فقدان بيانات |
| غضب معتدل | رؤية ظلم رقمي |
| مفاجأة | اكتشاف معلومة جديدة |
| فخر / ثقة | حماية نفسه أو مساعدة صديق |
| تصميم / عزم | مواجهة تحدي سيبراني |
| ارتباك | عدم فهم موقف رقمي |
| راحة / اطمئنان | الشعور بالأمان بعد اتخاذ قرار سليم |

## 10. اتفاقيات تسمية الشخصيات العربية

### قواعد التسمية
- اختر أسماء عربية شائعة وسهلة النطق
- المعنى يجب أن يرتبط بدور الشخصية
- تجنب الأسماء المرتبطة بشخصيات دينية مقدسة في أدوار سلبية
- اختر أسماء تعمل في مختلف اللهجات العربية
- وفر نطقا إنجليزيا بسيطا للأسماء (Transliteration)
