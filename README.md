## ما هو المشروع؟

**CyberPsych Kids** هو نظام إنشاء محتوى تعليمي متكامل يعمل من خلال Claude Code، مصمم لحماية الأطفال والشباب العرب في العالم الرقمي. يقدم المشروع محتوى توعوياً مبنياً على أسس علم النفس السيبراني، ومتوافقاً مع القيم الإسلامية والثقافة العربية.

يغطي النظام **10 فئات عمرية** من عمر سنتين وحتى 40 سنة، ويوفر **22 أمراً** لإنشاء قصص وألعاب وسيناريوهات فيديو وخطط دروس وأنشطة تفاعلية والمزيد.

---

## المميزات

- **10 فئات عمرية** تغطي جميع مراحل النمو من الطفولة المبكرة إلى الوالدية الرقمية
- **22 أمراً متخصصاً** لإنشاء أنواع مختلفة من المحتوى التعليمي
- **10 قوالب جاهزة** لتسريع عملية إنتاج المحتوى
- **14 ملف معرفة** تشكل القاعدة العلمية للنظام
- **منهجية مبنية على الأدلة** (Evidence-Based Cyber Psychology)
- **تكامل مع القيم الإسلامية** والثقافة العربية دون إقحام أو تكلف
- **مخرجات متعددة** تشمل HTML وMarkdown وJSON وPrompts
- **نمط القصة-الدرس-النشاط** لضمان تعلم فعال وممتع
- **دعم ثنائي اللغة** عربي وإنجليزي
- **تكامل مع أدوات الذكاء الاصطناعي** مثل Freepik وBlender MCP

## الفئات العمرية المدعومة

| الفئة العمرية | الاسم | المستوى | الأسلوب الأساسي |
|:---:|---|---|---|
| 2-4 سنوات | عالم الشاشة السحري | تمهيدي | ألوان وأشكال بسيطة، أغاني قصيرة |
| 4-6 سنوات | مغامرات في عالم الإنترنت | أساسي مبكر | قصص مصورة قصيرة، ألعاب بسيطة |
| 6-8 سنوات | فريق الأمان الرقمي | أساسي | قصص تفاعلية، أنشطة عملية |
| 8-10 سنوات | محققو الإنترنت | متوسط مبكر | مغامرات تحقيقية، تحديات ذهنية |
| 10-12 سنة | أبطال العالم الرقمي | متوسط | مشاريع جماعية، سيناريوهات واقعية |
| 12-14 سنة | الحياة الرقمية الذكية | متقدم مبكر | نقاشات، دراسات حالة، تحليل نقدي |
| 14-16 سنة | قادة العصر الرقمي | متقدم | قيادة أقران، مشاريع مجتمعية |

| 18-28 سنة | الحياة الرقمية الواعية | شباب | محتوى مهني، صحة نفسية رقمية |
| 28-40 سنة | الوالدية الرقمية والقيادة | والدين | أدلة إرشادية، أدوات مراقبة واعية |

## التثبيت

### 1. استنساخ المشروع

```bash
git clone https://github.com/your-username/cyberpsych-kids.git
cd cyberpsych-kids
```

### 2. فتح المشروع مع Claude Code

```bash
claude
```

سيقرأ Claude ملف `CLAUDE.md` تلقائياً ويُفعّل جميع الأوامر والقوالب.

### 3. البدء بإنشاء المحتوى

```bash
/story التنمر الإلكتروني --age 8-10 --output html --lang ar
```

```

```

## أمثلة استخدام

### إنشاء قصة عن الخصوصية الرقمية
```bash
/story الخصوصية الرقمية --age 6-8 --output html --lang ar
```

### تصميم لعبة عن كلمات المرور
```bash
/game-design كلمات المرور القوية --age 8-10 --output json --lang ar
```

### إعداد خطة درس عن التنمر الإلكتروني
```bash
/lesson التنمر الإلكتروني --age 10-12 --output html --lang both
```

### كتابة سيناريو فيديو عن البصمة الرقمية
```bash
/video-script البصمة الرقمية --age 14-16 --output markdown --lang ar
```
