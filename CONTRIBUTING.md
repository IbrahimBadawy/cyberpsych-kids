# دليل المساهمة - Contributing Guide

شكراً لاهتمامك بالمساهمة في مشروع CyberPsych Kids!

## كيف تساهم

### 1. إضافة أوامر جديدة (Commands)
- أنشئ ملف `.md` في `.claude/commands/`
- اتبع نفس تنسيق الأوامر الموجودة (frontmatter + sections)
- تأكد من دعم parameters: `--age`, `--output`, `--lang`

### 2. إضافة محتوى تعليمي
- أضف قصصاً أو أنشطة في `output/`
- تأكد من تحديد الفئة العمرية
- اتبع القوالب في `templates/`

### 3. إضافة ملفات معرفة
- أضف ملفات في `knowledge/`
- رقّم الملف بالترتيب
- وثّق المصادر

### 4. الترجمة
- ترجمة المحتوى للغات أخرى
- الحفاظ على المصطلحات التقنية الإنجليزية

### 5. تحسين القوالب
- تحسين القوالب في `templates/`
- إضافة أمثلة عملية

## قواعد المساهمة

1. **اللغة:** العربية الفصحى المبسطة + مصطلحات إنجليزية تقنية
2. **القيم:** المحتوى يجب أن يتوافق مع القيم الإسلامية
3. **الجودة:** كل محتوى يجب أن يكون مبنياً على أدلة علمية
4. **الملاءمة:** تحديد الفئة العمرية بوضوح
5. **التنسيق:** اتبع تنسيق الملفات الموجودة

## خطوات Pull Request

1. Fork المشروع
2. أنشئ branch جديد: `git checkout -b feature/اسم-المميزة`
3. أضف تغييراتك
4. اكتب وصف واضح للتغييرات
5. أرسل Pull Request

## الإبلاغ عن مشاكل
- افتح Issue على GitHub
- وصف المشكلة بوضوح
- اقترح حل إن أمكن

## التواصل
- Issues على GitHub
- المناقشات في Discussions
