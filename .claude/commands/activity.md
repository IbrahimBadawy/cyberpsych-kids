# نشاط تربوي - Activity Design

## متى تستخدم هذه المهارة
- إنشاء نشاط حركي يعلّم مفهوم سيبراني من خلال الحركة
- نشاط فني (رسم، تلوين، أشغال يدوية) عن الأمان الرقمي
- نشاط ذهني (ألغاز، تحديات، حل مشكلات)
- نشاط علاجي (CBT مبسط، يقظة ذهنية، تنفس، فن علاجي)

## المبدأ الأساسي
**النشاط الجيد يجعل الطفل يعيش التجربة بجسده وحواسه، لا أن يسمع عنها فقط.**

## المدخلات
- الموضوع (topic) - مطلوب
- --age [2-4|4-6|6-8|8-10|10-12|12-14|14-16|16-18|18-28|28-40] - مطلوب
- --type [physical|art|mental|therapeutic|mindfulness] - مطلوب
- --output [html|markdown|instructions|prompts] - افتراضي: markdown
- --lang [ar|en|both] - افتراضي: ar

## الخطوات

1. **اقرأ** knowledge/04-age-frameworks.md و knowledge/09-gamification-guide.md
2. **إذا كان علاجياً:** اقرأ knowledge/05-treatment-techniques.md

### هيكل النشاط
```
اسم النشاط: [اسم جذاب بالعربية]
النوع: [حركي | فني | ذهني | علاجي | يقظة ذهنية]
الفئة العمرية: [X-Y سنة]
المدة: [10-30 دقيقة]
عدد المشاركين: [فردي | ثنائي | جماعي]
المفهوم السيبراني: [المفهوم المراد تعليمه]
الهدف التعليمي: [ماذا سيتعلم المشارك]

المواد المطلوبة:
- [مواد متوفرة في المنزل/المدرسة]

الخطوات:
1. التحضير (2 دقيقة): ...
2. الشرح (3 دقائق): ...
3. التنفيذ (10-20 دقيقة): ...
4. المناقشة (5 دقائق): ...

تنويعات:
- نسخة أسهل للأصغر سناً
- نسخة أصعب للأكبر سناً
- نسخة جماعية

أسئلة نقاش بعد النشاط:
1. ماذا تعلمت؟
2. كيف تطبق هذا في حياتك؟

ملاحظات أمان:
- [أي احتياطات ضرورية]
```

5. **للأنشطة العلاجية** أضف:
   - التقنية المستخدمة (CBT/Mindfulness/Art Therapy)
   - تمارين التنفس (شهيق 4 ثوان، حبس 4، زفير 4)
   - تمارين إعادة الهيكلة المعرفية المبسطة
   - سجل المتابعة

## أمثلة
```
/activity "كلمات المرور القوية" --age 6-8 --type physical --output html
/activity "التنفس قبل الشاشة" --age 4-6 --type mindfulness --output instructions
/activity "رسم مشاعري مع الإنترنت" --age 8-10 --type art --output html
/activity "تحدي الانفصال الرقمي" --age 12-14 --type therapeutic --output markdown
```
