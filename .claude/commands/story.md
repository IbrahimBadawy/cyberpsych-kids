# قصة مصورة - Illustrated Story

## متى تستخدم هذه المهارة
- كتابة قصة تعليمية عن مفهوم سيبراني للأطفال
- إنشاء سلسلة قصصية متصلة
- تحويل مفهوم علمي معقد إلى قصة بسيطة

## المبدأ الأساسي
**القصة الناجحة تَغرس المفهوم في قلب الطفل دون أن يشعر أنه في درس. إذا شعر الطفل بالوعظ المباشر، فقد فشلت القصة.**

## المدخلات
- الموضوع (topic) - مطلوب: المفهوم السيبراني
- --age [2-4|4-6|6-8|8-10|10-12|12-14] - مطلوب
- --output [html|markdown|prompts] - افتراضي: markdown
- --lang [ar|en|both] - افتراضي: ar
- --scenes [3-5|5-8|8-12] - افتراضي: حسب العمر
- --theme [egyptian|gulf|levantine|maghreb|universal-arabic|universal] - افتراضي: egyptian

### الثيمات المتاحة
| الثيم | الأسماء | البيئة | اللهجة في الحوار |
|---|---|---|---|
| **egyptian** (افتراضي) | يوسف، مريم، عمر، حبيبة | مدارس مصرية، شوارع مصرية | فصحى مبسطة + تعبيرات مصرية |
| **gulf** | أمان، نورة، فارس، سديم | بيئة خليجية | فصحى مبسطة + تعبيرات خليجية |
| **levantine** | كريم، لين، سامر، رنا | بيئة شامية | فصحى مبسطة + تعبيرات شامية |
| **maghreb** | أمين، إيمان، ياسين، سلمى | بيئة مغاربية | فصحى مبسطة |
| **universal-arabic** | أسماء عربية عامة | بيئة عربية عامة | فصحى مبسطة فقط |
| **universal** | أسماء محايدة | بيئة عامة | عربية أو إنجليزية |

### هيكل القصة
```
العنوان: [عنوان جذاب]
الفئة العمرية: [X-Y]
المفهوم السيبراني: [مثل: التنمر الإلكتروني]
الهدف التعليمي: [جملة واحدة]

الشخصيات:
- البطل/ة: [اسم عربي، عمر، صفات]
- الصديق/ة: [المساعد في الحل]
- التحدي: [الموقف أو الشخصية المضادة]

المشاهد:
1. التقديم: تعريف البطل وعالمه
2. الموقف: ظهور المشكلة السيبرانية
3. التأثير: كيف أثرت على البطل (مشاعر)
4. البحث عن حل: محاولات البطل
5. الحل: اكتشاف الطريقة الصحيحة
6. الختام: الدرس المستفاد + فرحة

لكل مشهد:
- وصف المكان
- الحوار (بالعربية الفصحى المبسطة)
- المشاعر
- Image Prompt (لتوليد صورة المشهد)
```

6. **إذا output=prompts:** اكتب prompt مفصل لكل مشهد لاستخدامه في Freepik/DALL-E
7. **إذا output=html:** أنشئ صفحة HTML تفاعلية جميلة بالقصة
8. **أضف** نشاط مصاحب وأسئلة نقاش

## قواعد القصة
- لا وعظ مباشر - الدرس يأتي من الأحداث
- البطل يكتشف الحل بنفسه (لا يُلقَّن)
- نهاية إيجابية دائماً
- مشاعر حقيقية يتعاطف معها الطفل
- بدائل إيجابية وليس فقط تحذيرات

## أمثلة
```
/story "التنمر الإلكتروني" --age 8-10 --output html --lang ar
/story "التنمر الإلكتروني" --age 8-10 --output html --theme gulf
/story "كلمة السر القوية" --age 4-6 --scenes 5 --output prompts
/story "FOMO" --age 12-14 --output markdown --lang both
/story "الخصوصية" --age 6-8 --theme egyptian --output html
```
