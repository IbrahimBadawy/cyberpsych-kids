# مشروع جديد - New Project

## متى تستخدم هذه المهارة
- بدء مشروع محتوى جديد (سلسلة قصص، دورة تدريبية، حملة توعية، إلخ)
- تنظيم العمل في مراحل واضحة (مناقشة → تخطيط → تنفيذ)

## المبدأ الأساسي
**كل مشروع له بيت خاص فيه. المناقشات والخطط والمخرجات كلها في مكان واحد منظم.**

## المدخلات
- اسم المشروع (name) - مطلوب: مثل "سلسلة-التنمر" أو "دورة-الخصوصية"
- --type [series|course|campaign|workshop-series|game|book] - مطلوب
- --age [الفئة العمرية] - مطلوب
- --theme [egyptian|gulf|levantine|universal] - افتراضي: egyptian

## الخطوات

1. **أنشئ** فولدر المشروع في `projects/[اسم-المشروع]/`:

```
projects/[اسم-المشروع]/
├── README.md                    # ملخص المشروع
├── 01-discussion/               # مرحلة المناقشة
│   ├── notes.md                 # ملاحظات المناقشة
│   ├── research.md              # البحث والمصادر
│   └── ideas.md                 # الأفكار والمقترحات
├── 02-plan/                     # مرحلة التخطيط
│   ├── plan.md                  # الخطة التفصيلية
│   ├── timeline.md              # الجدول الزمني
│   └── resources.md             # الموارد المطلوبة
├── 03-implementation/           # مرحلة التنفيذ
│   ├── progress.md              # تتبع التقدم
│   └── [المحتوى المنتج]         # الملفات الفعلية
├── assets/                      # الأصول (صور، صوت، إلخ)
│   ├── images/
│   ├── prompts/                 # prompts للأدوات
│   └── references/              # مراجع
└── status.json                  # حالة المشروع
```

2. **أنشئ** `README.md` للمشروع:
```markdown
# [اسم المشروع]

| الحقل | القيمة |
|-------|--------|
| النوع | [series/course/campaign/...] |
| الفئة العمرية | [X-Y] |
| الثيم | [egyptian/...] |
| الحالة | 🟡 مناقشة |
| تاريخ الإنشاء | [التاريخ] |

## الفكرة
[وصف مختصر]

## المراحل
- [🟡] مناقشة
- [ ] تخطيط
- [ ] تنفيذ
- [ ] مراجعة
- [ ] نشر
```

3. **أنشئ** `status.json`:
```json
{
  "name": "[اسم المشروع]",
  "type": "[النوع]",
  "age": "[الفئة]",
  "theme": "[الثيم]",
  "phase": "discussion",
  "created": "[التاريخ]",
  "updated": "[التاريخ]"
}
```

4. **ابدأ** مرحلة المناقشة تلقائياً - اسأل المستخدم عن:
   - ما الهدف من المشروع؟
   - ما المفاهيم السيبرانية المراد تغطيتها؟
   - ما حجم المشروع المتوقع؟
   - هل يوجد مراجع أو إلهام محدد؟
5. **سجّل** كل النقاش في `01-discussion/notes.md`

## أمثلة
```
/project-new "سلسلة-قصص-التنمر" --type series --age 8-10
/project-new "دورة-الأمان-للمعلمين" --type course --age 28-40
/project-new "حملة-رمضان-الرقمية" --type campaign --age 10-14
/project-new "لعبة-محققو-الإنترنت" --type game --age 8-12
```
