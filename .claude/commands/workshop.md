# ورشة عمل تفاعلية - Workshop Design

## متى تستخدم هذه المهارة
- تصميم ورشة عمل للمدارس أو المراكز المجتمعية أو أونلاين
- تدريب معلمين أو والدين على الأمان السيبراني
- فعاليات توعوية للأطفال والشباب

## المبدأ الأساسي
**الورشة الناجحة تجعل المشارك يكتشف المعلومة بنفسه من خلال التجربة، لا أن يُلقّن إياها.**

## المدخلات
- الموضوع (topic) - مطلوب: مثل "التنمر الإلكتروني"، "كلمات المرور"
- --age [2-4|4-6|6-8|8-10|10-12|12-14|14-16|16-18|18-28|28-40] - مطلوب
- --output [html|markdown|slides|instructions] - افتراضي: markdown
- --lang [ar|en|both] - افتراضي: ar
- --theme [egyptian|gulf|levantine|maghreb|universal-arabic|universal] - افتراضي: egyptian
- --duration [2h|3h|4h] - افتراضي: 2h
- --mode [in-person|virtual|hybrid] - افتراضي: in-person

## الخطوات

1. **اقرأ** knowledge/04-age-frameworks.md لفهم الفئة العمرية
2. **اقرأ** knowledge/05-treatment-techniques.md للتقنيات المناسبة
3. **اقرأ** templates/workshop-template.md للقالب
4. **صمم** الورشة وفق هذا الهيكل:

### هيكل الورشة
```
أ. التحضير (قبل الورشة)
   - الأهداف التعليمية (3-5 أهداف SMART)
   - المواد المطلوبة
   - إعداد المكان/المنصة
   - عدد المشاركين المثالي

ب. كسر الجليد (10-15 دقيقة)
   - نشاط تعارف مرتبط بالموضوع
   - سؤال افتتاحي تفاعلي

ج. الجزء الأول: الاكتشاف (30-45 دقيقة)
   - عرض المشكلة بطريقة تفاعلية
   - نقاش جماعي
   - نشاط فردي أو ثنائي

د. استراحة (15 دقيقة)

هـ. الجزء الثاني: التطبيق (30-45 دقيقة)
   - نشاط عملي/لعبة تعليمية
   - عمل جماعي (مجموعات 3-5)
   - عرض نتائج المجموعات

و. الجزء الثالث: التأمل (15-20 دقيقة)
   - ماذا تعلمنا؟
   - خطة عمل شخصية
   - تعهد/التزام

ز. التقييم (10 دقيقة)
   - استبيان سريع
   - ملاحظات المشاركين
```

5. **أنشئ** المحتوى بالتفصيل لكل قسم
6. **أضف** تعديلات حسب الوضع (حضوري/افتراضي)
7. **اكتب** المخرج بالصيغة المطلوبة

## أمثلة
```
/workshop "التنمر الإلكتروني" --age 10-12 --duration 3h --output html
/workshop "خصوصية البيانات" --age 14-16 --mode virtual --output slides
/workshop "الوالدية الرقمية" --age 28-40 --output markdown --lang both
```
