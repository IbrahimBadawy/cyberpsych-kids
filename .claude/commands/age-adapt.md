# تكييف المحتوى لفئة عمرية - Age Adapt

## متى تستخدم هذه المهارة
- تحويل قصة من فئة 6-8 إلى فئة 10-12
- تبسيط درس للأطفال الأصغر
- تعقيد نشاط للمراهقين
- تكييف دليل والدين لفئة عمرية مختلفة

## المبدأ الأساسي
**التكييف ليس مجرد تغيير كلمات - هو إعادة تصميم التجربة بالكامل لتناسب مرحلة النمو.**

## المدخلات
- ملف المحتوى الأصلي (file path) - مطلوب
- --age [الفئة العمرية الجديدة] - مطلوب
- --output [نفس صيغة الأصلي] - افتراضي: نفس الصيغة
- --lang [ar|en|both] - افتراضي: نفس اللغة

## الخطوات
1. **اقرأ** المحتوى الأصلي
2. **اقرأ** knowledge/04-age-frameworks.md للفئتين (الأصلية والجديدة)
3. **حلل** الفروق بين الفئتين:
   - المفردات المناسبة
   - مستوى التعقيد المفاهيمي
   - طول المحتوى
   - نوع التفاعل
   - الأسلوب البصري
4. **عدّل** المحتوى:
   - للأصغر: بسّط المفردات، قصّر، أضف صور، قلّل المفاهيم
   - للأكبر: عمّق التحليل، أضف سياق، زد التعقيد، أضف تفكير نقدي
5. **راجع** بقراءة knowledge/06-content-guidelines.md
6. **اكتب** المحتوى المكيَّف

## أمثلة
```
/age-adapt output/stories/cyberbullying-6-8.md --age 12-14
/age-adapt output/lessons/passwords.md --age 4-6
/age-adapt output/activities/digital-detox.md --age 28-40
```
