# دليل الوالدين - Parent Guide

## متى تستخدم هذه المهارة
- إنشاء دليل للوالدين عن مشكلة رقمية محددة
- تحضير مادة لورشة عمل للأهالي
- إنشاء نشرة تثقيفية للوالدين

## المبدأ الأساسي
**الوالد المثقف رقمياً هو أفضل حماية لطفله. لا نخيف الوالدين بل نمكّنهم.**

## المدخلات
- الموضوع (topic) - مطلوب
- --age [عمر الطفل المعني] - مطلوب
- --output [html|markdown|instructions] - افتراضي: markdown
- --lang [ar|en|both] - افتراضي: ar

## الخطوات
1. **اقرأ** knowledge/03-digital-problems.md و knowledge/05-treatment-techniques.md
2. **اقرأ** knowledge/12-screen-time-guidelines.md
3. **اقرأ** templates/parent-guide-template.md
4. **اكتب** الدليل:

### هيكل الدليل
```
ما يجب أن تعرفه (3-5 نقاط):
- تعريف المشكلة ببساطة
- لماذا تحدث
- حجم المشكلة (إحصائيات)

علامات التحذير (قائمة مراجعة):
□ العلامة 1
□ العلامة 2
□ ...

بدايات محادثة مع طفلك (5 أسئلة):
1. "حبيبي، ..."
2. ...

خطوات عملية (مرقمة):
1. الخطوة الأولى
2. ...

نشاط عائلي مقترح:
[نشاط يعمله الأهل مع الطفل]

موارد إضافية:
- روابط ومصادر

متى تطلب مساعدة متخصصة:
- الحالات التي تستدعي استشارة نفسية
```

## أمثلة
```
/parent-guide "إدمان الألعاب" --age 8-10 --output html
/parent-guide "ابني يتعرض للتنمر الإلكتروني" --age 12-14 --output markdown
/parent-guide "وقت الشاشة المناسب" --age 2-4 --lang both
```
