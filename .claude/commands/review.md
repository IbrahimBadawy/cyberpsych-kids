# مراجعة المحتوى - Content Review

## متى تستخدم هذه المهارة
- مراجعة قصة أو درس قبل النشر
- التأكد من ملاءمة المحتوى للفئة العمرية
- فحص التوافق مع القيم الإسلامية والثقافية
- تقييم الجودة التعليمية والتفاعلية

## المبدأ الأساسي
**المحتوى الجيد يجتاز 4 اختبارات: مناسب للعمر، متوافق ثقافياً، مفيد تعليمياً، ممتع وجذاب.**

## المدخلات
- ملف المحتوى (file path) - مطلوب
- --output [html|markdown] - افتراضي: markdown
- --lang [ar|en|both] - افتراضي: ar

## الخطوات
1. **اقرأ** المحتوى المراد مراجعته
2. **اقرأ** knowledge/04-age-frameworks.md و knowledge/14-islamic-cultural-guide.md
3. **قيّم** وفق 6 معايير (كل معيار من 10):

### بطاقة التقييم
```
═══════════════════════════════════
       بطاقة تقييم المحتوى
═══════════════════════════════════
المحتوى: [اسم الملف]
الفئة العمرية المستهدفة: [X-Y]

1. ملاءمة العمر:        [_/10]
   - المفردات مناسبة؟
   - التعقيد مناسب؟
   - الطول مناسب؟

2. التوافق الثقافي:     [_/10]
   - القيم الإسلامية محترمة؟
   - لا يوجد محتوى حساس؟
   - الأسماء والسياق عربي؟

3. القيمة التعليمية:    [_/10]
   - الهدف التعليمي واضح؟
   - المعلومات دقيقة علمياً؟
   - مبني على أدلة؟

4. الجاذبية والتفاعل:   [_/10]
   - ممتع وشيق؟
   - يتضمن تفاعل؟
   - يحفز التفكير؟

5. الإيجابية:           [_/10]
   - يقدم بدائل إيجابية؟
   - لا يعتمد على التخويف؟
   - يمكّن لا يُضعف؟

6. دقة المعلومات:       [_/10]
   - المصطلحات صحيحة؟
   - الإحصائيات محدثة؟
   - لا توجد معلومات مضللة؟

═══════════════════════════════════
المجموع:                [_/60]
التقدير: [ممتاز|جيد جداً|جيد|يحتاج تحسين]
═══════════════════════════════════
```

4. **اكتب** اقتراحات التحسين المحددة
5. **حدد** الأولويات (عاجل/مهم/اختياري)

## أمثلة
```
/review output/stories/fomo-story-10-12.md
/review output/lessons/digital-detox.html
```
