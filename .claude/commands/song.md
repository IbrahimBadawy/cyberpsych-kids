# أغنية/نشيد تعليمي - Song/Nasheed

## متى تستخدم هذه المهارة
- كتابة أغنية تعليمية عن مفهوم سيبراني للأطفال
- إنشاء نشيد يمكن ترديده في المدرسة أو البيت
- كتابة أناشيد لفيديوهات تعليمية

## المبدأ الأساسي
**الأغنية تُثبّت المعلومة في الذاكرة طويلة المدى. إذا حفظها الطفل سيتذكر الدرس سنوات.**

## المدخلات
- الموضوع (topic) - مطلوب
- --age [2-4|4-6|6-8|8-10|10-12] - مطلوب (الأغاني للأطفال حتى 12)
- --output [html|markdown|prompts] - افتراضي: markdown
- --lang [ar|en|both] - افتراضي: ar
- --theme [egyptian|gulf|levantine|maghreb|universal-arabic|universal] - افتراضي: egyptian
- --style [nasheed|song|rap|chant] - افتراضي: nasheed

## الخطوات
1. **حدد** الفئة العمرية والمفهوم السيبراني
2. **اختر** الإيقاع المناسب للعمر:
   - 2-4: بطيء، تكراري، 4-8 أسطر
   - 4-6: إيقاعي متوسط، قافية واضحة، 8-12 سطر
   - 6-8: حيوي، مع لازمة، 12-16 سطر
   - 8-10: متنوع الإيقاع، مع مقاطع وكورَس
   - 10-12: يمكن أن يكون rap تعليمي

3. **اكتب** الكلمات بالعربية الفصحى المبسطة:
   - قافية واضحة وسهلة
   - كلمات بسيطة يفهمها الطفل
   - تكرار اللازمة (الكورَس)
   - دمج المفهوم بشكل طبيعي

4. **أضف** الحركات المصاحبة:
   - حركات يد لكل سطر
   - تمثيل بالجسم للمفاهيم
   - تصفيق أو إيقاع بالأرجل

5. **أضف** وصف الإيقاع الموسيقي

## أمثلة
```
/song "كلمة السر القوية" --age 4-6 --style nasheed --lang ar
/song "لا تتحدث مع الغرباء" --age 6-8 --style song --output html
/song "أنا أتحكم في شاشتي" --age 8-10 --style rap --lang both
```
