# لعبة أسرية - Family Game

## متى تستخدم هذه المهارة
- تصميم لعبة يلعبها الأهل مع أطفالهم
- إنشاء بطاقات لعب قابلة للطباعة
- تصميم لعبة لوقت السيارة أو العشاء
- ألعاب خارجية تعلّم مفاهيم سيبرانية

## المبدأ الأساسي
**اللعبة الأسرية تحقق هدفين: تعلّم الأمان السيبراني وتعزيز الترابط العائلي. الأهم هو أن يضحك الجميع.**

## المدخلات
- الموضوع (topic) - مطلوب
- --age [الفئة العمرية لأصغر لاعب] - مطلوب
- --type [board|card|outdoor|dinner|car|quick] - مطلوب
- --players [2-4|3-6|4-8] - افتراضي: 3-6
- --output [html|instructions|markdown] - افتراضي: instructions
- --lang [ar|en|both] - افتراضي: ar

### هيكل اللعبة الأسرية
```
🎲 اسم اللعبة: [اسم جذاب]
👨‍👩‍👧‍👦 اللاعبون: [العدد]
⏱️ المدة: [15-45 دقيقة]
🎯 المفهوم السيبراني: [المفهوم]
📦 المواد المطلوبة: [أشياء موجودة في البيت]

═══ القواعد ═══
التحضير: ...
طريقة اللعب: ...
الفوز: ...
التنويعات: ...

═══ أسئلة نقاش بعد اللعبة ═══
1. ما الذي تعلمناه؟
2. كيف نطبق هذا في حياتنا الرقمية؟
3. ما الذي فاجأكم؟
```

### أنواع الألعاب:
- **لوحية (board):** لوحة + بطاقات + نرد + أحجار
- **بطاقات (card):** بطاقات فقط (سهلة الطباعة)
- **خارجية (outdoor):** ألعاب حركة في الحديقة/الفناء
- **عشاء (dinner):** ألعاب على مائدة الطعام (لا تحتاج مواد)
- **سيارة (car):** ألعاب شفهية للسفر
- **سريعة (quick):** 5-10 دقائق

4. **إذا HTML:** صمم بطاقات/لوحة قابلة للطباعة
5. **أضف** نصائح للوالدين لتسهيل اللعب

## أمثلة
```
/family-game "كلمات المرور" --age 6-8 --type card --output html
/family-game "التنمر الإلكتروني" --age 8-10 --type board --players 3-6
/family-game "من يعرف أكثر عن الإنترنت" --age 10-12 --type dinner
/family-game "سباق الأمان الرقمي" --age 6-8 --type outdoor --output instructions
```
