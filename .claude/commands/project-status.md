# حالة المشاريع - Project Status

## متى تستخدم هذه المهارة
- عرض حالة مشروع محدد
- عرض قائمة كل المشاريع
- التنقل بين المشاريع
- معرفة أين توقفت في مشروع

## المدخلات
- اسم المشروع (project) - اختياري (بدون اسم = عرض كل المشاريع)
- --action [status|list|switch|archive] - افتراضي: status

## الخطوات

### إذا بدون اسم مشروع (عرض الكل):
1. **امسح** مجلد `projects/` للمشاريع الموجودة
2. **اقرأ** `status.json` من كل مشروع
3. **اعرض** جدول ملخص:

```
═══════════════════════════════════════════════════
              المشاريع النشطة
═══════════════════════════════════════════════════

| # | المشروع | النوع | العمر | المرحلة | التقدم |
|---|---------|-------|-------|---------|--------|
| 1 | سلسلة-قصص-التنمر | series | 8-10 | 🔄 تنفيذ | 3/8 |
| 2 | دورة-المعلمين | course | 28-40 | 📋 تخطيط | - |
| 3 | حملة-رمضان | campaign | 10-14 | 💬 مناقشة | - |

═══════════════════════════════════════════════════
للعمل على مشروع: /project-status "اسم-المشروع"
لإنشاء مشروع جديد: /project-new "اسم" --type [نوع] --age [عمر]
═══════════════════════════════════════════════════
```

### إذا مع اسم مشروع (عرض التفاصيل):
1. **اقرأ** `projects/[المشروع]/status.json`
2. **اقرأ** `projects/[المشروع]/README.md`
3. **اقرأ** آخر ملفات المرحلة الحالية
4. **اعرض** تفاصيل المشروع:

```
═══════════════════════════════════════════════════
         مشروع: سلسلة-قصص-التنمر
═══════════════════════════════════════════════════

المرحلة الحالية: 🔄 تنفيذ (3/8 مكتمل)
الفئة العمرية: 8-10 سنوات
الثيم: مصري

المراحل:
  ✅ مناقشة (مكتمل - 2026-04-13)
  ✅ تخطيط (مكتمل - 2026-04-14)
  🔄 تنفيذ (جاري - 3/8)
  ⏳ مراجعة
  ⏳ نشر

العناصر:
  ✅ 1. قصة "يوسف والرسالة الغريبة"
  ✅ 2. قصة "مريم والتعليق المؤذي"
  ✅ 3. قصة "عمر والصورة المسروقة"
  🔄 4. قصة "حبيبة والصفحة المزيفة"
  ⏳ 5-8. ...

الأوامر المتاحة:
  /project-discuss "سلسلة-قصص-التنمر"  ← للمناقشة
  /project-plan "سلسلة-قصص-التنمر"     ← للتخطيط
  /project-start "سلسلة-قصص-التنمر"    ← لاستكمال التنفيذ
═══════════════════════════════════════════════════
```

### إذا --action archive:
- انقل المشروع لمجلد `projects/_archive/`
- حدّث status.json → `"phase": "archived"`

## أمثلة
```
/project-status
/project-status "سلسلة-قصص-التنمر"
/project-status "دورة-المعلمين" --action archive
```
