# مراجع الأدوات

## اشتراكات مدفوعة
- **Claude Code:** skills, commands, MCP tools, artifacts
- **Freepik Spaces:** node-based AI workflow - صور + فيديو + صوت + Custom Characters
- **ChatGPT:** DALL-E, Custom GPTs, Canvas
- **Gemini:** Imagen, NotebookLM, AI Studio, TTS, Long Context 1M

## أدوات مجانية رئيسية
- **Scratch (scratch.mit.edu):** ألعاب تعليمية للأطفال 8+
- **GDevelop (gdevelop.io):** ألعاب بدون كود - مثبت على الجهاز
- **Twine (twinery.org):** قصص تفاعلية (choose your adventure)
- **CapCut:** محرر فيديو مجاني - مثبت على الجهاز
- **Canva Free:** تصميم + قوالب تعليمية
- **Leonardo.ai:** 150 صورة/يوم مجاناً
- **Ideogram:** أفضل أداة لنص داخل الصور
- **ElevenLabs Free:** 10 دقائق صوت/شهر
- **Figma Education:** مجاني بالكامل للمعلمين

## منصات نشر مجانية
- YouTube Kids, TikTok, Instagram
- Amazon KDP (كتب), itch.io (ألعاب)
- Spotify for Creators (بودكاست)
- Kahoot, Google Classroom (تعليم)

## MCPs متاحة
- Blender MCP (Hyper3D, Hunyuan3D, Polyhaven, Sketchfab)
- NotebookLM connector (عبر Chrome)
- Claude Preview (HTML preview)
