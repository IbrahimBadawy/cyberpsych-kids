## خطوات التنفيذ في Freepik Spaces

### الخطوة 1: إنشاء Space جديد
1. افتح freepik.com/spaces
2. اضغط "Create new Space"
3. سمّيه "يوسف والشاشة الجائعة"

### الخطوة 2: إنشاء Character Reference
1. أضف Image Generator node
2. الصق prompt يوسف → ولّد
3. احفظ كـ Custom Element
4. كرر لمريم والشاشة الجائعة

### الخطوة 3: توليد المشاهد
1. أضف Image Generator node لكل صفحة
2. الصق prompt الصفحة
3. استخدم Custom Element ليوسف لضمان الاتساق
4. ولّد واختر أفضل نتيجة

### الخطوة 4: تحسين الجودة
1. استخدم Image Upscaler (2x أو 4x)
2. تأكد من اتساق الألوان

### الخطوة 5: إضافة النص (في Canva)
1. حمّل الصور من Spaces
2. افتح Canva → قالب كتاب أطفال
3. أضف الصور + النص العربي
4. صدّر كـ PDF
